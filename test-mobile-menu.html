Mobile Menu Test Page

This page is for testing the mobile hamburger menu functionality.

Instructions:

  1. Resize your browser window to mobile size (or use developer tools)
  2. Click the hamburger menu button (three lines) in the top right
  3. The menu should expand/collapse
  4. The icon should change between hamburger and X
  5. Check browser console for debug messages

Open browser developer tools and check the Console tab for debug messages.
